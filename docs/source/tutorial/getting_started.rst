Getting started
===============

The remainder of this tutorial uses examples shipped with ``nvtest``.  To obtain a copy of the examples, execute the following command in your shell:

.. code-block:: console

    $ nvtest fetch examples

.. note::

    ``nvtest`` must be on your ``PATH``
